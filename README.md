# PetitPetri

PetitPetri is a A Petri Net Editor built with Etoys.

See the [Petit Petri web page](http://scg.unibe.ch/download/petitpetri/) for details.

This repo contains all the source files for that page, except for the [prepared image](http://scg.unibe.ch/download/petitpetri/petitpetri-squeakjs/PetitPetriEToys.image) for [SqueakJS](https://squeak.js.org).

These files can be downloaded from [github](https://github.com/onierstrasz/PetitPetri).
